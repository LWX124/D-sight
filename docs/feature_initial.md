当前文档是项目的初始需求文档，具体的需求如下：

# 1. 入口是聊天界面

    可以根据用户的输入进行股票分析、当日最新信息总结、可套利基金等（后续新的支持的需要可扩展）

# 2. skill

    - 有 skill 市场，里面不可自定义上传，而是存放经过筛选的skill
    - 默认为大家安装 ai-berkshire 中的全部skill，可以进行股票的分析
    ai-berkshire 的源码在 /Users/weixi1/Documents/Study/ai-berkshire

# 3. 支持知识库

    - 支持用户上传文档变成自己的知识库
    - 支持加载共享知识库
    - 知识库本身需要共享功能

# 4. 新闻热点

    这个板块主要分为三部分：
    （1） 7 x 24 小时快讯，这里可以定时5min去获取新浪财经的接口数据存到我们的数据库，（接口后续提供，先写逻辑）
    （2）支持其他信源信息获取（可添加信源是我这边提供的），后面对接一些信源板块
    （3）社媒信息，这里支持小红书、微博、微信公众号、哔哩哔哩等媒体渠道的信息获取（这里面每一个的添加和信息获取后续会逐个完善，没有通用方案）

# 5. 注册/登录系统

    - 邮箱注册登录
    - 微信的单点登录

# 6. 积分/订阅

    - 后面有些功能需要消耗积分，默认每个月给100积分（月初重置）
    - 订阅的每个月给2000积分

# 7. 需要LLM 的判断，需要LLM provider能力，暂时集成 deepseek
